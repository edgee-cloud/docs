[Iubenda](https://www.iubenda.com/) provides a comprehensive consent management solution to help you comply with global privacy regulations and manage user preferences.

This component automatically handles consent management by integrating directly with Iubenda, eliminating the need to [manually specify consent](/proxy/services/consent/overview) through the Edgee SDK or Data Layer.

The consent status will be automatically set to `pending`, `granted`, or `denied` based on the Iubenda cookie.

## Getting Started

**To integrate Iubenda Consent Mapping into your Edgee project:**

1. Open the Edgee console and navigate to your project's Components.
2. Select "Add a component" and choose `edgee/iubenda-consent-mapping` from the list of available components.
3. Add your Iubenda site ID in the component settings.
4. Activate the component on the top right.
5. Confirm with "Add component" to complete the setup.

## Where to find the Iubenda site ID?

There is two ways to find the Iubenda site ID:

1. From the Iubenda dashboard
2. In the browser console, when you are on the website you want to configure

### From the Iubenda dashboard

You can find the Iubenda site ID in the Iubenda dashboard.

1. Open the Iubenda dashboard.
2. Click on the website you want to configure
3. Look at the url, and you'll see something like this: https://www.iubenda.com/fr/flow/123456#/
4. The site ID is the value of the "123456" part.

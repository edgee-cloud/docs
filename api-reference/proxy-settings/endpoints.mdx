---
title: "ProxySettings endpoints"
description: "ProxySettings endpoints allow you to list, create and update the proxy settings of a Project."
---

A ProxySettings object represents a revision of a proxy configuration of a Project.


<RequestExample>
```js ENDPOINTS
GET /v1/proxy-settings
POST /v1/proxy-settings
POST /v1/proxy-settings/:revision
```
</RequestExample>

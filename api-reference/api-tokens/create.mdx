---
title: "Create an ApiToken"
description: "Creates an ApiToken for the current user. ApiTokens are used to authenticate with the Edgee API.
You can create as many api tokens as you want."
---

### Parameters

<ResponseField name="name" type="string" required>
  This is the name of your api token. It is used to identify the token when you list your api tokens.
</ResponseField>

<ResponseField name="expires_at" type="string">
  This is the expiration date of the api token. If not provided, the token will never expire.
</ResponseField>


### Response

Returns the created api token if successful. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash POST /v1/api_tokens
curl 'https://api.edgee.app/v1/api_tokens' \
-u '<token>:' \
-d 'name=my very secret token' \
-d 'expires_at=2032-02-20T00:00:00Z'
```
</RequestExample>

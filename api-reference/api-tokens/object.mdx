---
title: "The ApiToken object"
description: "The ApiToken object is used to represent an API token in Edgee."
---

### Attributes

<ResponseField name="id" type="string">
  This is the internal ID for this API token. You don't need to record this
  information, since you will not need to use it.
</ResponseField>

<ResponseField name="object" type="string">
  String representing the objectâ€™s type. Objects of the same type share the same value.
</ResponseField>

<ResponseField name="user_id" type="string">
  The ID of the user that created this API token.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the API token.
</ResponseField>

<ResponseField name="from_browser" type="boolean">
  Whether this API token was created from the browser or not.
</ResponseField>

<ResponseField name="last_used_at" type="string">
  The last time this API token was used.
</ResponseField>

<ResponseField name="expires_at" type="string">
  The datetime this API token will expire.
</ResponseField>

<ResponseField name="created_at" type="string">
  The datetime at which this API token was created.
</ResponseField>

<ResponseField name="updated_at" type="string">
  The datetime at which this API token was last updated.
</ResponseField>

<ResponseField name="token" type="string">
  The actual token that you will use to authenticate your requests.
  
  <Info>Be careful, this token is only returned once, when the token is created.</Info>
</ResponseField>

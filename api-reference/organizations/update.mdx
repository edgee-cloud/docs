---
title: "Update an Organization"
description: "Update an Organization for the current user."
---

### Parameters

<ResponseField name="name" type="string">
  The name of the organization.
</ResponseField>

<ResponseField name="slug" type="string">
  The slug of the organization.
</ResponseField>

### Response

Returns the updated Organization object if successful. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash POST /v1/organizations/:id
curl 'https://api.edgee.app/v1/organizations/752268f6-102a-40c8-b726-0a5abe891083' \
-u '<token>:' \
-d 'name=My Company Name' \
-d 'slug=my-company-name'
```
</RequestExample>

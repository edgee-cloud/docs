---
title: "User endpoints"
description: "User endpoints allow you to get, invite and list Users."
---

A user is a person who can log in to Edgee. A user can be a member of one or more organizations.

You can invite as many users as you like to your organization. You can also invite users to multiple organizations.


<RequestExample>
```js ENDPOINTS
GET /v1/users/me
GET /v1/users/:id
```
</RequestExample>

---
title: "ProjectComponent endpoints"
description: "ProjectComponent endpoints allow you to create, list, get and delete Component for a project."
---

A ProjectComponent is an integration of a component for a project. 


<RequestExample>
```js ENDPOINTS
POST /v1/projects/:id/components
POST /v1/projects/:id/components/:project_component_id
GET /v1/projects/:id/components/:project_component_id
DELETE /v1/projects/:id/components/:project_component_id 
GET /v1/projects/:id/components
```
</RequestExample>

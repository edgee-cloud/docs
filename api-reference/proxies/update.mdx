---
title: "Update a Proxy"
description: "Update a Proxy for the current user."
---

### Parameters

<ResponseField name="name" type="string">
  The name of the proxy.
</ResponseField>

<ResponseField name="description" type="string">
  The description of the proxy.
</ResponseField>

<ResponseField name="organization_id" type="string">
  The organization ID of the proxy.
</ResponseField>

### Response

Returns the updated Proxy object if successful. Otherwise, this call raises an [error](../errors).


<RequestExample>
```bash POST /v1/proxies/:id
curl 'https://api.edgee.app/v1/proxies/fca28702-2ce9-45e0-ad21-e9320aa08731' \
-u '<token>:' \
-d 'name=My amazing proxy' \
-d 'description=This is my amazing proxy' \
```
</RequestExample>

<ResponseExample>
